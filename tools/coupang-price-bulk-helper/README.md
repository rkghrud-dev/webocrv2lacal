# Coupang Price Bulk Helper

쿠팡 Wing 상품목록 화면에서 리포트 기준 가격 인하 작업을 보조하는 Chrome 확장프로그램입니다.

## 설치

1. Chrome 주소창에 `chrome://extensions` 입력
2. 우측 상단 `개발자 모드` 켜기
3. `압축해제된 확장 프로그램을 로드` 클릭
4. 이 폴더 선택:
   `C:\Users\rkghr\Desktop\WEBOCRV2_LOCAL\tools\coupang-price-bulk-helper`

## 사용

1. 쿠팡 Wing 상품목록 페이지를 새로고침합니다.
2. 우측 하단 `쿠팡 가격도우미` 패널에서 `현재 화면 매칭`을 누릅니다.
3. `선택+입력`을 누르면 화면에 보이는 첫 매칭 행을 체크하고 판매가 변경창에 인하 금액을 입력합니다.
4. 기본값은 저장 전 멈춤입니다. 확인 후 쿠팡 모달에서 직접 `저장`을 누르세요.
5. 저장까지 자동 실행하려면 패널에서 `저장까지 자동 실행`을 체크한 뒤 사용합니다.

## 주의

- 확장프로그램은 쿠팡 서버 제한을 우회하지 않습니다.
- 현재 화면에 렌더링된 상품 행만 처리합니다.
- 쿠팡 UI가 바뀌면 버튼/모달 탐지가 실패할 수 있습니다.
- `저장까지 자동 실행`은 실제 판매가를 변경하므로 소량으로 테스트 후 사용하세요.
